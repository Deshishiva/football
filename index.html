Goal Shock Trading Engine

Submission for Omniverse Fund – AI/ML / Quant Intern Assessment. This page provides a clear overview of the project and links to the source code.

1. Project Summary

This engine listens to simulated real-time football goal events and reacts when an underdog team scores a goal. The strategy attempts to capture short-lived price mispricing in prediction markets such as Kalshi and Polymarket by placing fast trades before markets adjust.

2. What the System Does

  • Receives goal updates (mock live feed)
  • Normalizes every event into a clean format
  • Maps events to a synthetic in-play market
  • Simulates market prices
  • Decision engine evaluates whether to BUY
  • Logs all orders and decisions

3. Source Code

Full project repository: https://github.com/Deshishiva/football-

4. How to Run Locally

git clone https://github.com/Deshishiva/football-
cd football-

# install dependency
pip install websocket-client

# run engine
python main.py

This starts the mock event feed and displays all trades in the terminal.

5. Workflow (Short Explanation)

  • MockFeed → Generates random match events
  • Normalizer → Cleans event structure
  • Matcher → Maps match to market ID
  • PriceFeed → Produces temporary market prices
  • Trader → Decides BUY / NO TRADE
